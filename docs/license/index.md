## 制作物について

### **個人用途**

個人的な用途ならば使用・解析・改変などは自由です。

### **再配布**

Chestnutの制作物の一部または全体は、対象人数に関わらず再配布が可能です。その際はこのライセンスの提示または同包をお願いします。

### **改変**

改変すること自体は可能ですが、改変物を配布する際は[ご相談](#ご連絡先)ください。

### **著作権**

Chestnutの著作権はGarnetが有しますが、Chestnutを使って制作された著作物はその制作者が有します。
